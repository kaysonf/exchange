The main aim of this project is to handle real time data on the front end

The backend does not implement any business logic - it only mocks real time data

# HOW TO RUN

## Server

```
$ cd server

$ npm i

$ npm run start
```

## Web

```
$ cd web

$ npm i

$ npm run start
```
